# Team Goal Tracker

A simple, accessible tracker website for teams to list goals and update progress with add/subtract buttons.

## Getting started

1. Open `index.html` in a browser.
2. Click **Add goal** to create a goal.
3. Use **+** or **–** to update completed tasks.
4. The summary updates automatically and data is saved in browser storage.

## Features

- Accessible keyboard support
- Clear goal list with progress counters
- Local persistence via `localStorage`
- Optional image URL support for goal cards
- Edit goal name, image, and target after creation
- Delete goals from the tracker
- Edit the page title
- Edit page colors with a live theme editor- Upload a background image file for the page and adjust its scaling
- Change whole-screen text styling (font, size, bold, italic, underline)- Per-goal progress rings to track percent complete
- Progress circle showing overall completion
- Add and subtract controls for each goal
